== Description ==

WP Hacker News is a simple plugin that will place a "Submit to Hacker News" icon
on each of your pages.  Once installed, each of your posts will have this icon in the
upper right-hand corner.  It is only viewable when viewing an individual post, as not to
clutter up site.

== Installation ==

1. Upload `wp-hacker-news.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy!

== Frequently Asked Questions ==

= Is there a way to change the size of the icon? =

Not yet.  Future revisions will allow you to set the size of the icon yourself.

= Can I remove the text below the icon? =

If you feel like editing the source you can.  Otherwise, the next revision will allow for this.
